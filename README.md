Python code accompanying Braam & Angerhausen (2025, in press at A&A), see https://arxiv.org/abs/2512.16619.
Repository contains Jupyter Notebooks that were used to convert the CCM data into binary format for PSG GlobES tool and to calculate the orbital evolution of Proxima Centauri b in 1:1 and 3:2 spin-orbit resonance.
Due to the large size of the output from the 3D coupled climate-chemistry models, the data will be shared by reasonable request to the corresponding author (Marrick Braam, marrick.braam@unibe.ch). The emission spectra were produced using NASA's Planetary Spectrum Generator and its GlobES tool (https://psg.gsfc.nasa.gov/apps/globes.php), and are provided in the folder emission_spectra_psg. We then use LIFEsim (https://lifesim.readthedocs.io/en/latest/) to generate synthetic LIFE spectra that
can be found in emission_spectra_LIFE.

**Reference**:
Braam and Angerhausen, 2026, Observing spatial and temporal variations in the atmospheric chemistry of rocky exoplanets: prospects for mid-infrared spectroscopy, A&A, 707, A281, https://doi.org/10.1051/0004-6361/202557807
